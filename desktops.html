The Linux Desktop

Software components should be small,
and focused on doing one specific thing well.
UNIX Philosophy

A package is a logical unit of software. The obvious example is a program, but libraries are packages, too. Packages may exist in either source-code or precompiled form. The purpose of this article is to orient the user to the package-centric landscape of pragmatic open-source software.
