Expressive

Rust is an expressive language. That means:

  1. It's enormous
  2. (Optionally) Everything is an expression

Everything in Rust really is an expression, and evaluates to some value. The none value is (), the unit tuple; this is the return type of a function which returns nothing, for instance, or, the value of an if statement.

"() is a value of the type () and its purpose is to be useless," says A.B. the Wise. It is an object with size of zero (bytes); and, it stands to reason that its existence would have no effect on machine state.
