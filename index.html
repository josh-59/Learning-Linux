Welcome!

Welcome to Learning Linux!

This website is intended to introduce the subject of Linux. Mostly, it is a collection of resources about the subject. In addition, it's a collection of writings which, together, cover most basic concepts. While intended for the new user, these writings do not shy away from rigorously developing the subject. Enjoy!

Conventions:

  • Links are restricted to those that are both useful and non-obvious
    • Inline links point out immediately relevant sources
      • i.e., they're worth looking into
    • Superscript references[1] denote authoritative resources which aren't necessarily of interest
      • i.e., don't read them, unless you must
    • Wikipedia is considered authoritative (and often, its documentation proves quite good)
  • Code and terminal commands are written in this font (Ubuntu Mono Bold)
    • Commands that may be entered are written on a new line.
    • The command-line prompt is represented with "$ " for user-level permissions and "# " for root-level permissions:
      $ command
