Pro Tip


Find Files with Grep:

grep -rl . -e PATTERN

Grep will search each file from "here" downwards, and, if the contents of that file match PATTERN, print the name of the file.

Option Summary:
  • -r Recursively descend through directory tree
  • -l List files; do not print matching line
  • . Begin at current directory
  • -e What follows is a PATTERN
