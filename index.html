Welcome to Learning Linux!

This website is intended to introduce the subject of Linux. Mostly, it is a collection of resources about the subject. In addition, it's a collection of writings which, together, cover most basic concepts. While intended for the new user, these writings do not shy away from rigorously developing the subject. Enjoy!

Link Convention:

Links are restricted to those that are both useful and non-obvious.

  • Inline links point out immediately relevant sources
    • Are worth looking into
  • Superscript[1] references denote authoritative resources which aren't necessarily of interest
    • Don't read them, unless you must

Wikipedia will be considered authoritative (and often, its documentation proves quite good)
