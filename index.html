Command-Line Pro Tip


Find man pages by section:

apropos -s 3 printf

Apropos will restrict its search to the manual section given.

Option Summary:
  • -s list, --sections list, --section list
    Search only the given manual sections.
    list is a colon- or comma-separated list of sections.

Source: apropos(1)
